Class RequiredValidator

Validates the presence of a required value

# __construct( boolean $strict = false )

Constructs a required validator

Constructs a required validator

If $strict is true, then when the value is present but equal to an empty value (for the PHP definition of emptiness, see empty()), the validator will pass as true.

However, if strict is false, we will also test that the value provided does not evaluate to false. If it does, then validation will fail.

By default, this does not do a strict check, so values that evaluate to boolean false will fail validation as a required value.
